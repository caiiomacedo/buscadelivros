# 📚 Busca de Livros

## 📖 Sobre o projeto
Este é um projeto web desenvolvido para realizar a busca de livros utilizando a API pública da Open Library.

A aplicação permite que o usuário pesquise livros pelo nome e visualize informações como título, autor, ano de publicação e quantidade de edições.

Os dados pesquisados são armazenados localmente no navegador (LocalStorage), permitindo que permaneçam disponíveis mesmo após recarregar a página.

---

## 🚀 Funcionalidades
- 🔍 Busca de livros por nome
- 📊 Exibição dos resultados em tabela
- 💾 Armazenamento local dos dados (LocalStorage)
- ❌ Remoção de livros da lista
- ⚠️ Validação de campo obrigatório com alerta visual
- 🎨 Interface estilizada com CSS

---

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- API Open Library
- SweetAlert2 (alertas personalizados)
- Alert padrão do navegador (tratamento de erro)

---

## ▶️ Como executar o projeto

1. Faça o download ou clone este repositório
2. Abra a pasta do projeto
3. Execute o arquivo `index.html` em qualquer navegador
